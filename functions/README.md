# PickLT Cloud Functions

This folder contains all the cloud functions for the **PickLT** project.

## Overview

All serverless cloud functions related to the PickLT application are organized and maintained within this directory.

## Getting Started

1. Navigate to this directory:

    ```bash
    cd /home/elisha/Desktop/projects/pickLT/functions
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Deploy the functions:

    ```bash
    npm run deploy
    ```
